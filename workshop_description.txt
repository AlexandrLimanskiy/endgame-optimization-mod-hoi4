[b]Endgame Performance Optimization[/b] — мод против просадки скорости в эндгейме HOI4 (1943+). При разработке мода значения были сняты с реальных дефайнов 1.19.2, а узкие места подтверждены встроенным профайлером игры.

Принципы: ни одна существующая дивизия никогда не удаляется, игрок ни в чём не ограничен, ванильные файлы не перезаписываются. Мод растягивает частоты фоновых пересчётов движка, убирает мусорные объекты из симуляции и мягко останавливает раскармливание армий ИИ сверх разумного предела.

[b]Почему игра тормозит к 1944+[/b]
[list]
[*] Каждая из ~100+ стран ИИ регулярно пересчитывает роли, морские цели, узкие места снабжения, авиаприоритеты, рынок вооружений — стоимость растёт квадратично с числом юнитов и фронтов
[*] Полный пересчёт всех торговых маршрутов мира каждые 45 дней
[*] Сотни устаревших шаблонов дивизий в матчинге производства у каждой страны
[*] ~310 ванильных событий, чьи триггеры движок опрашивает по каждой стране и штату
[*] Страны-призраки (остатки гражданских войн: 0 штатов, 0 войск) тикают как полноценные страны
[/list]

[b]Что делает мод[/b]

[u]Дефайны симуляции[/u] — растянуты частоты фоновых пересчётов ИИ (роли, морские цели, снабжение, авиаприоритеты, рейды, исследования, рынок), в первую очередь параметры, помеченные самими Paradox как «affects performance». Торговые маршруты: полный пересчёт 45 → 90 дней. Устаревшие шаблоны дивизий удаляются в 4 раза быстрее. Цикл опроса событий 20 → 30 дней. Крупнее флотские таскфорсы ИИ (меньше объектов с почасовым тиком), смена боевых тактик раз в сутки, реже проходы распределения подкреплений. Каждое значение снабжено ванильным оригиналом в комментарии.

[u]Контроль армий ИИ[/u] — потолок желаемых дивизий 500 → 250 плюс регулятор с обратной связью: чем сильнее страна превысила порог (250/300/350), тем сильнее давится тренировка новых дивизий — это гасит в том числе ванильную стратегию раскармливания СССР («FEED THE MEATGRINDER»). Существующие дивизии не удаляются, избыток рассасывается фронтовой убылью. Потолок общий для всех ИИ — баланс сил сохраняется. Пресеты мягче/жёстче правятся в две строки в дефайнах.

[u]Сборщик мусора[/u] — раз в месяц удаляются страны-призраки: 0 штатов, 0 дивизий/флота/авиации, в мире, не марионетка, не правительство в изгнании, не в фракции, только ИИ. Им нечего передавать, поэтому геймплей не меняется — освобождаются только пустые тики.

[u]Панель управления[/u] — вкладка решений «Оптимизация производительности»: ручной запуск глубокой очистки (кулдаун 7 дней), вкл/выкл автоочистки, счётчик удалённых стран-призраков.

[u]Агрессивный режим (опция, по умолчанию выключен)[/u] — включается решением в игре: ежемесячная утилизация гор устаревших винтовок 0–1 уровня на складах ИИ, уже массово владеющих современными (до 5000/мес с гарнизонным резервом; игрока не касается).

[u]Графика[/u] — дальность отрисовки погодных частиц 1500 → 1000 (вблизи камеры погода полная).

[u]В файле дефайнов закомментированы два опциональных блока[/u] — ускорение мирных конференций и вдвое меньшая генерация асов: раскомментируются одной строкой, по умолчанию выключены, т.к. слегка влияют на геймплей.

[b]Совместимость[/b]
[list]
[*] Версия игры: 1.19.*
[*] Не перезаписывает ни один ванильный файл — совместим с большинством модов (конфликты возможны только по одинаковым дефайнам, побеждает нижний в плейсете)
[*] DLC не требуется
[*] Мультиплеер: работает, мод должен стоять у всех игроков
[*] Меняет контрольную сумму → достижения Ironman недоступны
[*] Можно включать/выключать на уже идущем сейве
[/list]
